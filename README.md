# 债券收益率曲线分析平台

## 功能概述

本平台用于分析债券收益率期限结构曲线，主要功能：

1. **数据预处理**：从Excel读取债券池，调用Choice API获取估值数据
2. **曲线拟合**：对每个发行主体拟合期限结构曲线（Nelson-Siegel或Hermite插值）
3. **特征提取**：计算曲线斜率、R²等特征指标
4. **分类分析**：区分普通债和永续债
5. **结果导出**：支持Excel导出

## 快速开始

双击 `启动服务器.bat`，或运行：

- 前端页面：http://localhost:8000/frontend/index.html

### 3. 更新数据

双击 `更新数据.bat`，选择更新方式：
1. 更新最新交易日
2. 更新指定日期
3. 批量更新日期范围

## 数据更新说明

### 是否需要每日运行？

**是的**，建议每日收盘后运行一次更新程序。

原因：
- 债券剩余期限每日变化
- 中债估值每日更新
- 新发行/到期债券需要更新

### 历史数据预计算

运行以下命令预计算历史数据：

```bash
python data_preprocess.py --start 2025-01-01 --end 2026-03-06
```

预计处理时间：8-12小时（282个交易日）
